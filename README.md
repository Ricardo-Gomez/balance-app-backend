## Description
Backend for the pet project to register incomes and outcomes.


[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Dependencies
MongoDB
NodeJS > 14
## Installation

```bash
$ npm install
```

## Setup
copy .env.example to .env and fill it up
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
